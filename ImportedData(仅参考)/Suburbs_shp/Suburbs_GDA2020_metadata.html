Description

DescriptionSuburbs and Localities (or Rural suburbs) are defined areas of the State established for addressing purposes. The base data used for the capture is the cadastre. The Suburbs layer is a data set that reflects the official boundaries of addressable localities in South Australia as defined by the Geographical Names Act, 1991. The polygons are based on the existing cadastral, topographic or road centerline data that defines the underlying features that boundaries follow.
UsageUsed to spatially define gazetted Suburb / Locality boundaries.

Data Quality

LineageThe Suburb Boundaries in the urban and rural areas are updated when required through notification in the Government Gazette or spatial improvement of the cadastre. The capturing started 1990 and is ongoing.
CompletenessThe Suburb boundary dataset is currently 100% complete for the state and complies with government classification standards.
Positional AccuracySuburb Boundaries are acquired by digitising using the cadastral boundaries as a reference. Consequently, the positional accuracy of the Suburb Boundaries are dependent upon the positional accuracy of the cadastre. In certain areas the positional accuracy of the cadastre has been improved from graphical accuracy to survey-accuracy.
